The path that contains "wsclient.jar" and "wsobjects.jar", if not specified, it will be auto-detected by locating "fortifyclient" in the search PATH, and then deduce the lib path. Both "fortifyclient", "wsclient.jar" and "wsobjects.jar" are included in the Fortify 360 installation

This path should be a local path of Hudson MASTER node.

If you don't have a full Fortify 360 installation on the Hudson MASTER node, you can just copy the following JARs and setup this path manually

  • wsclient.jar
  • wsobjects.jar
  • common.jar (only for v2.6 or above)
  • common13.jar (only for v2.6 or above)
